# Run the project :

```
cd frontd/
git init
npm i
```

Frontend structure

- components: atoms, molecules, organisms, templates
- features: all slices (redux toolkit)
- pages: project pages
- routes
- store: configuration of redux store
- styles: general styles scss
- translations
- types

To add a new route:

Update useAdminRoutes.tsx or useUserRoutes.tsx
Update MenuItems.tsx
