# Mobile App Implementation Plan

## Structure Kotlin à créer:
1. LoginActivity.kt - écran de connexion
2. TransporterActivity.kt - espace transporteur
3. ClientActivity.kt - espace client
4. API Service - connection à https://api.samlogistic.tn

## Étapes:
1. Convertir MainActivity en Kotlin
2. Créer les écrans avec Jetpack Compose ou XML
3. Implémenter l'authentification
4. Builder l'APK

→ Cette implémentation prendra plusieurs heures. L'utilisateur doit utiliser Android Studio pour ouvrir le projet et continuer le développement.
